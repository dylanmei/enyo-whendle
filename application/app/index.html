Whendle is an alarm-clock application to manage multiple alarms. Whendle includes an exhibition mode to display the current time.

This is an early release version of the TouchPad edition of Whendle. Some edges are still rough, and many of the features available in the original Whendle are still on their way.

The default alarm is a sound from freesound.org: Fire Alarm by Benboncan.
